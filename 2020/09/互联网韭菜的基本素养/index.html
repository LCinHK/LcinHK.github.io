    互联网韭菜的自我修养

    发表于 2020-09-21 分类于 日常 阅读次数：

    社会网络韭菜三大件，火狐电报n++。本文简要分享下个人网络生活惯用技巧

    浏览器——上网之本

    不管在什么时候，能用浏览器**100%**比使用客户端好。因为javascript是唯一一个不触及系统的语言。
    主用火狐国际版，辅助用chrome canary。强烈推荐的理由如下↓

    1. 火狐国际版的扩展市场国内可用，而chrome商店目前是被挡住的。装不了扩展的chrome就是半残废
    2. 火狐国际版尤其developer版本对前端开发友好。
    3. 原生搭载广告与矿工拦截器。
    4. Firefox LockWise可以用来代替密码管理器。
    5. 辅助用chrome以调试一些会被拦截的部分，如广告。

    为什么不用国内一些主流浏览器，如三百六、两千三百四十五、对Q等？

    理由很简单：

    1. 所有的功能都能被火狐浏览器替代。（需要”双内核“的场合为什么不用真ie呢？）
    2. 大部分的主流浏览器是基于chrome和火狐所用内核打包的。同一家族选孙子不如选爷爷。
    3. 对Q等浏览器会无端屏蔽网站，如996.icu 的项目主页
    4. 扩展商店非标准化
      此外，国产免费软件会有着众所周知问题：广告和隐私问题。毕竟这是他们主要的盈利变现渠道。

    自定义化你的浏览器

    这是目前我在用的扩展。从左到右依次为：

    • 在哪都要https ——除了安全问题以外，强制开启https对登陆一些没做301跳转的网站（如163）很有用。
    • 你挡源 ——ublock origin是一款相对来说好用亲民的广告拦截器。具有自定义功能，可以显著提升速度。
    • 查IP ——由北京天特信科技有限公司研发，用来简单看下网站IP与位置信息。
    • 单HTML下载 ——传统的Control + S 下载网页HTML往往会需要多一个用于存放css和图片等静态文件的文件夹，这款可以直接将所有需要内容整进一个HTML文件里
    • 切换UA ——有时候，你需要换成不同的浏览器和系统User Agent（如测试移动端网页）
    • 编辑Headers ——主要作用就是在某些环境把google CDN换成镜像。
    • JetBrains插件 ——其实就是给github、gitlab等网站添加一个一键在jetbrains家 IDE开启的功能
    • archive保存网页 ——防削除！将网页存档到archive.today上。

    此外，建议在任何浏览器任何模式下设置“关闭浏览器时清理一切”。cookie和历史记录既占空间又不安全。

    网络设置

    DNS

    不建议国内用户改默认DNS，原因是会破坏CDN的作用。此外，大家耳熟能详的一一四DNS有记录用户数据和投送分地域广告的历史。如果因为环境（如公司内网）需要设置DNS，推荐设定为离你最近的公共DNS服务(ping一下看延时)或CNNIC的SDNS。任何情况都不要在大陆使用境外DNS，慢得不行。

    IP

    公网IP很有用。如何辨别自己是否有公网IP条件？

    1. 如果直接电脑连接，请进行ipconfig，如下图↓
      其中，IPv6 Address为2开头而非fe80证明我有公网IPv6地址，而IPv4地址为192.168开头证明我没有公网IPv4地址，可以看看位于192.168.1.1的网关有无公网IPv4地址
    2. 进入宽带网关看看获取到的IP地址是否为内网IPv4。如果获取到结果和随意哪个查IP网站(如北京天特信)一致，那么必定是正常的公网IPv4

    没有公网IPv4地址的可以尝试拨打宽带运营商电话（如长城宽带95079、中国联通10086）然后强调要改公网IP地址。
    有了公网IP地址你将可以：

    1. IP直连联机游戏
    2. 访问内网假设的服务器、NAS云存储等
    3. 高速上下载BT种子和ed2k
    4. 不受大内网总带宽限制
    5. 防止出现“您的IP已经投过票/注册过/被标记为高危”

    云存储服务

    强烈推荐尽快逃离百度网盘。百度不仅会扫描文件并替换8秒，而且所有文件均不会消失。无法彻底删除（哪怕是重要隐私）文件。

    Onedrive

    众所周知，只要你使用微软windows10系统onedrive就会预载。而经过一段时间的使用后，笔者在此强烈推荐onedrive，理由如下：

    1. 文件安全。完全没有”极速秒传“等困扰。
    2. 方便快捷。onedrive相当于原生的webdav支持网盘，平常可以方便存取。
    3. 作为服务器位于境外的网盘，网速相对够用。
    4. 和office等微软家产品联动很好。附带分享office文档功能

    需要注意的是，onedrive有不同版本。以下除特别注明世纪互联版皆为国际版。

    特性 个人版 家庭版 学校/企业版 世纪互联版
    网页端 连接已重置 连接已重置 可以正常打开，网速较慢 正常
    在线分享文件 连接已重置 连接已重置 体验很好，支持很多功能 不了解
    网速 一般 一般 较快 理论上最快
    和微软产品联动 完整 完整 一部分可用 不太行
    售价 ￥400/人/年 ￥500/6人/年 问你学校/公司 仅有企业版，最低￥50/人/月
    容量 1TB 每人1TB 1-5TB不等，取决于你学校/公司如何 1-5TB

    其他

    • 可以参考一些原本多用于家用游戏机的教程设置MTU。通过ping -f -l 14xx blog.alevel.tech（xx请换成两位数，并确定不会出现错误的最大值）来确定应该设置为多少
    • 建议开个网速监控器看看是否有软件当前正占据大带宽
    • 网线的稳定性是Wi-Fi所不能比拟的，但速度不是。
